Homepage of the WebServer Example

The following pages are available:

  • /index.htm - This page
  • /files.htm - Manage files on the server
  • /$upload.htm - Built-in upload utility
  • /none.htm - See the default response when files are not found.

The following REST services are available:

  • /api/sysinfo - Some system level information
  • /api/list - Array of all files
